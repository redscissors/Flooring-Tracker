# Skills reference (retired skill library)

These folders were the project's `.claude/skills/` library until 2026-07-14.
They were moved here so `.claude/skills/` could host the general-purpose
superpowers workflow skills (github.com/obra/superpowers) instead.

They are **no longer skills** — nothing auto-triggers them and the old slash
commands (`/decide`, `/design-review`, `/handoff`, …) are gone. Treat each
`SKILL.md` as ordinary project documentation: when its topic comes up (change
control, estimate math, debugging, price-book domain knowledge, …), read the
relevant file like any other doc.

The `floortrack-*` knowledge packs remain the deepest written record of how
this app works and why; CLAUDE.md links to the load-bearing ones.
